
## Резюме диаграмм

| Диаграмма | Что показывает | Для кого |
|-----------|----------------|----------|
| **1. Component** | High-level обзор системы | Архитекторы, PM |
| **2. Domain Model** | Основные сущности | Разработчики |
| **3. Repository** | Repository Pattern | Backend devs |
| **4. Service Layer** | Бизнес-логика | Backend devs |
| **5. Platform Adapter** | Адаптеры платформ | Разработчики интеграций |
| **6. Ban Flow Sequence** | Процесс обработки бана | Все |
| **7. Subscription Flow** | Создание подписки | Backend devs |
| **8. Package Dependency** | Зависимости модулей | Архитекторы |
| **9. Deployment** | Инфраструктура | DevOps |
| **10. Database ER** | Схема БД | DB admins, Backend |
